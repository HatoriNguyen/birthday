# 🎉 Chúc Mừng Sinh Nhật Đoàn Vương Khang (2009) - Gen Z Birthday Web Page

Trang web chúc mừng sinh nhật được xây dựng với giao diện hiện đại, phong cách neon glassmorphic, nhiều hiệu ứng động vui tươi thích hợp cho thế hệ Gen Z và tích hợp âm nhạc sôi động.

## 🌟 Tính năng nổi bật

1. **Màn hình chào mừng (Welcome Gate):** Người dùng bấm vào nút "Mở hộp quà" để kích hoạt âm nhạc và mở thiệp sinh nhật, vượt qua hạn chế chặn autoplay nhạc của trình duyệt.
2. **Nhạc nền Đinh Tùng Happy Birthday:** Nhúng bài hát "Khúc hát mừng sinh nhật" do Phan Đinh Tùng thể hiện (YouTube Video ID: `WjV56-hJ-Rk`) thông qua YouTube Iframe API.
3. **Đĩa nhạc quay (Vinyl disc spinner):** Đĩa nhạc quay tròn khi nhạc phát và dừng quay khi nhạc tạm dừng. Có bộ điều khiển Play/Pause dễ thương.
4. **Bánh kem thổi nến tương tác:** Bánh kem 3 tầng có 3 ngọn nến đang cháy. Click vào nến để thổi tắt, khi tắt hết nến sẽ bắn pháo hoa ngập trời và hiện lời chúc mừng.
5. **Cơn mưa Emoji & Pháo hoa Canvas:** Bong bóng và emoji chúc mừng liên tục bay từ dưới màn hình lên. Khi bấm chuột vào các vị trí trống trên màn hình, pháo hoa rực rỡ sẽ nổ tại đó.
6. **Thanh sticker Gen Z:** Chứa các cụm từ hài hước ("mận keo", "flex", "sos", "cột sống", "hot"). Khi bấm vào sticker sẽ hiện thông báo độc lạ và bắn thêm emoji chúc mừng.
7. **Bảng lời chúc Carousel:** Tự động chuyển đổi hoặc bấm nút thủ công giữa các lời chúc sinh nhật tuổi 17 cực "mận keo".

## 🚀 Hướng dẫn chạy thử cục bộ

Bạn chỉ cần nhấp đúp vào tệp tin **`index.html`** để mở trực tiếp trên các trình duyệt web hiện đại (Google Chrome, Microsoft Edge, Safari, Firefox).

## 🌐 Triển khai lên GitHub Pages

Để biến trang web này thành một đường link trực tuyến gửi cho Đoàn Vương Khang:

1. Đẩy mã nguồn lên kho lưu trữ GitHub của bạn:
   ```bash
   git add .
   git commit -m "feat: init gen z birthday web app for Doan Vuong Khang"
   git branch -M main
   git push origin main
   ```
2. Trên trang web GitHub của repo (`https://github.com/HatoriNguyen/birthday`), đi tới mục **Settings** -> **Pages**.
3. Tại phần **Build and deployment** -> **Source**, chọn **Deploy from a branch**.
4. Tại phần **Branch**, chọn nhánh **`main`** và thư mục **`/ (root)`**, sau đó nhấn **Save**.
5. Đợi 1-2 phút, GitHub sẽ cung cấp một đường link truy cập công khai (ví dụ: `https://HatoriNguyen.github.io/birthday/`).
